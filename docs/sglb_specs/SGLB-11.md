# SGLB-11 Citation-Hallucination

Version: 0.1-shipped (smoke). Tracking issue: [#52](https://github.com/gongahkia/junas/issues/52).

## Capability

**C3 — Citation handling (hallucination detection).** Given a passage
containing real and injected-fake SG citations, identify the fakes.

## Literature anchor

- Li et al., **HaluEval** (EMNLP 2023, [arXiv:2305.11747](https://arxiv.org/abs/2305.11747))
  — injection methodology.
- Dahl et al., **Large Legal Fictions** (J. Legal Analysis 2024,
  [arXiv:2401.01301](https://arxiv.org/abs/2401.01301)) — strongest
  empirical baseline.

## Input contract

```python
case.inputs = {
  "passage": str,           # 300–600-token passage with N citations
}
```

The passage is told to the model: *"List the citations in this passage
that are fabricated. Output a JSON array of citation strings."*

## Output contract

Model output is a JSON list of citation strings the model flags as
fake:

```json
["[2025] SGCA 999", "Acme Corp v Globex Inc [2020] SGHC 50"]
```

## Scoring

- **Per-citation P/R/F1:** treat the citation set as a binary
  classification (real vs fake). F1 is the headline.
- **Per-perturbation breakdown:** F1 stratified by perturbation type
  (`year_off`, `volume_off`, `page_off`, `case_name_swap`, `court_swap`,
  `wholesale_fabrication`, `composite`).
- **False-flagging rate:** real citations marked fake. Practically
  important.

## Source provenance

- Adapters: `SsoAdapter`, `ElitigationAdapter`, `CommonliiSgAdapter`
  for the real-citation pool. Fakes generated by perturbation rules.
- `verifier.py` cross-checks every fake against the real pool to ensure
  no accidental collisions; documented false-fake rate ≤1%.

## Limitations

- Year-off perturbations may accidentally be real cases; `verifier.py`
  re-rolls collisions.
- Models with web-search ability solve trivially; the leaderboard flags
  web-access setting per row and reports closed-book scores as primary.
- Dataset contamination from training is mitigated by post-2026-Q1
  held-out split.

## v0.1 / v0.2 stratification

- v0.1 smoke: 40 passages, 7 perturbation classes all represented,
  ~60-citation real pool. Deterministic (seed=42). Shipped at
  `backend/benchmark/datasets/sglb_11_hallucination_smoke.yaml`.
- v0.2 production: ~500 passages, ~2000 distinct citations, ~50/50
  real/fake by citation count.
- v0.2 held-out: ~100 passages built from post-2026-Q1 real citations.
- Per-perturbation leaderboard breakdown is mandatory.
