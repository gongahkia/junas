# SGLB-04 Citation-Verify

Version: 0.1-shipped. Tracking issue: [#29](https://github.com/gongahkia/junas/issues/29), [#32](https://github.com/gongahkia/junas/issues/32).

## Capability

**C3 — Citation handling.** Given a candidate SG legal citation,
determine whether it conforms to the SAL Style Guide grammar.

## Literature anchor

- The SAL Style Guide grammar itself (Quick Reference 2007 +
  SLR Style Guide 2021) is the authority.
- Guha et al., **LegalBench** (NeurIPS 2023), `citation_extraction`
  family — but our evaluator is deterministic, not LLM-judge.

## Input contract

```python
case.inputs = {
  "citation": str,   # candidate citation, e.g. "[2023] SGCA 5"
}
```

## Output contract

Model output is a single-element JSON list containing `"valid"` or
`"invalid"`:

```json
["valid"]
```

## Scoring

- `multi_label_f1` over the single-element labels list with
  `expected_output["labels"]` set to `["valid"]` or `["invalid"]`.
- Strict mode (`--strict`) required for publication runs.
- Per-case evaluator details carry `metadata.breakdown`, so receipts can
  be grouped by `stratum`, `perturbation`, `validation_kind`, and
  `expected_errors` for per-error reporting.

## Source provenance

- No external adapter needed. Dataset constructed mechanically from the
  SAL grammar implementation in `api.services.sal_citation`.
- Production dataset:
  `benchmark/datasets/sglb_04_citation_verify_full.yaml`, generated by
  `benchmark/dataset_builders/sglb_04.py`.
- Smoke dataset:
  `benchmark/datasets/sglb_04_citation_verify_smoke.yaml` (30 cases,
  preserved for fast local testing). The legacy
  `benchmark/datasets/sglb_04_citation_verify.yaml` path remains as the
  original smoke fixture for existing tests and scripts.
- Perturbation taxonomy aligns with SGLB-11:
  - `year_off`, `volume_off`, `page_off`, `case_name_swap`, `court_swap`,
    `wholesale_fabrication`, `composite`.

## Limitations

- Grammar conformance ≠ existence. A perfectly-formed `[2026] SGCA 999`
  passes SGLB-04 but may not be a real case. SGLB-11 covers existence.
- The grammar reflects the 2007 + 2021 guides; pre-2007 citation forms
  may legitimately fail and are excluded from the dataset.

## v0.1 stratification

- Production set: 1,080 cases.
- Invalid perturbation strata: 120 cases each for `year_off`,
  `volume_off`, `page_off`, `case_name_swap`, `court_swap`,
  `wholesale_fabrication`, and `composite`.
- Valid negative controls: 240 cases labelled `valid` with
  `stratum=valid_negative`.
- Split policy: `train` / `dev` / `test` assigned 80/10/10 by SHA-256
  hash of the case name. Current generated counts are train=872,
  dev=109, test=99.
- Error-code occurrences reportable from evaluator receipt rows:
  `year_out_of_range` 240, `volume_non_positive` 180,
  `page_non_positive` 180, `unknown_court` 180,
  `case_no_non_positive` 60, `no_grammar_match` 240. Composite cases
  intentionally contribute more than one error code.
