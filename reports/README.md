# Reports Folder

This folder stores generated outputs rather than source code.

- `layer-attribution/`: committed candidate evaluation and miss-attribution evidence used by docs/tests.
- `current/`: standalone committed summaries generated from reviewed candidate/evaluation workflows.
- `current/ai4privacy_pii_masking_200k_en_us_en_gb_eval.json`: semi-independent ai4privacy breadth eval over documented US/UK English proxy slices.
- `historical/latency/`: retained historical latency text snapshots.
- Root-level `latency_*.json`, `latency_*.csv`, `latency_slo_*.json`, and backend logs may be generated by local or CI benchmark runs.
